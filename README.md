Voodoo
------

! WIP !

A NewSQLy Database of sorts.

Declarative DDL - the entire database structure is kept in a single file,
any changes are "applied".

# Datatypes

Voodoo has a relatively restricted set of primitive
datatypes.

- TIMESTAMP
- TIMEZONE
- DATE 
- INT64
- FLOAT64
- TEXT
- BOOL
